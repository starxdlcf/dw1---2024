Como fazer listas no html

Ordenadas

As listas ordenadas são classificadas com números, letras e outras coisas que são elementos únicos (aparecem uma vez só, pois caracterizam um elemento específico) - funciona como se fosse uma pk de banco de dados.

Para fazer uma lista ordenada você utiliza a tag ol e coloca as características dela no inline

Exemplos:

  1. alface
  2. tomate
  3. feijão
  4. batata
  1. banana
  2. morango
  3. caju
  4. abacaxi
