Pesquisa sobre algumas tags HTML

Code

A tag code serve para identificar um código dentro do seu arquivo html. A parte do texto que estiver entre as tags de código será mostrada de maneira monoespaçada e um pouco diferente das outras partes do seu texto, justamente para caracterizar aquela parte do texto como sendo um trecho de código.

Exemplo:

Este texto contém um código... Você consegue saber onde ele está? Dica: a tag code foi utilizada nele.

Pre

A tag pre significa texto pré-formatado, e ela indica que o texto escrito dentro dela será mostrado no site da mesma forma que foi escrito: com todos os enters e espaçamentos que estão presentes.

Exemplo:

        Este texto contém     mu  u  u  u  u  u  u  ito        espaço       entre      as         palavras.
        E um enter bem
        aqui e
        aqui!

Samp

A tag samp é utilizada para denotar um trecho de um resultado impresso por um computador (uma saída). Sua fonte geralmente é a padrão do navegador e é monoespaçada.

Exemplo:

Hoje eu fui ligar meu computador e apareceu a seguinte mensagem: Verifique se seu computador está na tomada; pois a bateria dele estava acabando.

Kbd

A tag kbd serve para denotar uma entrada de voz, teclado ou quaisquer outro tipos que possam ser realizados pelo usuário. Geralmente essa tag é mostrada com uma fonte padrão e de forma monoespaçada, mas isso não é uma regra.

Exemplo:

Meus atalhos favoritos!!!

  1. Aperte ! + enter para colocar a base do arquivo HTML;
  2. Aperte ctrl + ; para indicar que a linha é um comentário;
  3. Aperte ctrl + shift + p para abrir uma barra de pesquisa de comandos (extensões);
  4. Aperte alt + z para impedir que as linhas do seu código fiquem muito longas e não apareçam inteiramente na sua tela.
